Discord Cloner V1 is a modern Discord server backup and migration tool designed for educational purposes. It allows users to create backups of their own Discord servers, restore saved data, and organize server structures through an easy-to-use interface.

# Quick Start

## Requirements

- Docker (optional)
- Python 3.11+
- Discord Bot

## Installation

```bash
git clone https://github.com/real3ethereum/Discord-Cloner-V1.git

cd Discord-Cloner-V1

pip install -r requirements.txt

python main.py
```
